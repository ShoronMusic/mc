# NotebookLM 用：洋楽AIチャット 宣伝素材まとめ

このファイルは **Google NotebookLM** にソースとして追加し、**15秒CM風動画**の構成案・ナレーション・テロップ案を生成させるための材料です。実装の内部名（リポジトリ名など）は宣伝文では使わず、利用者向けの言い方に寄せています。

---

## 1. サービス概要（NotebookLM に渡す説明文）

- **名称**: 洋楽AIチャット（β版）※UI・案内では「洋楽AIチャット」で統一してよい（仮称の扱いあり）。
- **一言**: 洋楽好きが集まり、**YouTube の曲を部屋で共有しながら同時に聴き、チャットで語れる** Web サービス。会話の流れに合わせて **AI が曲まわりや豆知識をサポート**する。
- **公式サイト（CTA）**: `https://musicai.jp`（SNS・プロフィールで案内している URL）
- **運営の姿勢**: 利用者の声を踏まえて改善中のベータ。マナー・AI の性質についてはサイト内の「ご利用上の注意」「利用規約」を参照。

---

## 2. 誰向けか（ターゲット）

- 洋楽（海外ポップ・ロック等）が好きで、**同じ曲を誰かと同時に聴きながら感想を言いたい人**。
- **YouTube にある曲**を軸に、部屋で流して盛り上がりたい人。
- 選曲や話題が途切れがちなので、**AI に軽くフォローしてほしい人**（押し付けではなく「サポート」として伝える）。
- 初めてでも、**URL を貼る・入室する**操作から始められる人。

---

## 3. 魅力・差別化（宣伝で使いやすいポイント）

以下は **利用者が体感しやすい順**に並べています。動画では上位数個に絞るとよいです。

1. **同時視聴＋チャット**  
   部屋で YouTube を共有し、**みんなで同じタイミングで聴きながら**その場で会話できる（「聴きながら語る」が主役）。

2. **YouTube URL で選曲**  
   馴染みの YouTube リンクを貼って選曲。**洋楽ファンにとって入口のハードルが低い**。

3. **AI が「曲まわり」をサポート**  
   選曲の流れや話題作り、曲にまつわる**わかりやすい補足**など、会話が続くよう手助けするイメージ（詳細機能名は伏せてよい）。

4. **AI への質問は `@` で**  
   発言の先頭に `@` を付けて **洋楽・音楽に関する質問**ができる（例: `@ おすすめの洋楽を1つ教えて`）。**音楽以外は想定外**であることも案内用に一言入れられる。

5. **登録ユーザーがルームを設置**  
   「自分の部屋を持つ」イメージ。ゲストは参加中心（設置は登録者向け）— **CMでは短く「部屋を作って招待」程度**で十分。

6. **（任意・上級者向け）Chrome 拡張**  
   YouTube から発言欄へ渡す補助がある（メイン訴求にしなくてよい。興味層向けの一文で可）。

## 4. あったらいいカット（画像・画面録画のシーン案）

NotebookLM に「この順で15秒に圧縮して」と頼むときの **カットリスト**として使えます。**スクショ1枚＝1カット**でも、画面録画の **タイムコード** でも可。

| 優先 | カット内容 | 画面上で見せるもの | 狙い |
|:---:|:---|:---|:---|
| A | サービス名・β | ロゴまたはトップのサービス名＋「β」 | 0.5秒で何のサイトか固定 |
| B | 部屋の「いま再生中」 | プレイヤー＋再生が動いている状態 | **同時視聴**が伝わる |
| C | URL 貼り付け→反映 | 入力欄に YouTube URL → 選曲される一連 | **入口が簡単** |
| D | チャットが流れる | ユーザー発言が複数行 | **語れる・ライブ感** |
| E | AI の短い返答がチャットに出る | `@` 付き質問 or 選曲まわりの AI 一行 | **AIサポート**が視覚的に分かる |
| F | 複数人っぽい UI | 参加者表示・アバター等があれば | **みんなで** |
| G | 締め CTA | `musicai.jp` を大きく（指差しアイコン任意） | 記憶に残すドメイン |

**避けた方がよい画**: 個人情報・メール通知・未読だらけの OS 画面。宣伝用には **専用ブラウザプロファイル**やデモ用アカウント推奨。

## 7. トーン・表現のガイド

## 9. 外部画像・動画素材と NotebookLM

本 MD は **テキスト中心**のソース用。ロゴ・背景・画面キャプチャ・ループ素材などの **画像・動画ファイルは、NotebookLM にソースとして個別にアップロード**して渡す（本書に **外部素材サイトの URL は記載しない**）。

### リポジトリ内 `videosozai/` の画像（意味付け）

NotebookLM にファイルを追加するとき、次の対応で解釈させられる。

| ファイル（リポジトリ上のパス） | 用途・イメージ |
|--------------------------------|----------------|
| `videosozai/cover03.png` | チャットに集う **ユーザー像**（雰囲気・イメージ画） |
| `videosozai/hukidasi.png` | チャットに飛び交う **会話の吹き出し**（会話・ライブ感のビジュアル） |
| `videosozai/music_ai_chat_logo.png` | **サイトロゴ**（タイトル・締め・ブランド認知用） |
| `videosozai/screen_01.png` | **実際のサイト画面**（UI・機能の説明に直結） |

- **規約・クレジット**: 各ファイルの入手元の条件に従う。必要なら、その規約ページの写し（PDF 等）を別ソースとして NotebookLM に追加する。
- **制作の割り振り**: フリー素材は **背景・転場・テロップ下地** に寄せ、**サービス説明の主役は自社の画面録画**（セクション4のカット）にするとよい。
- **クレジット表記が必須**の素材は、動画末尾テロップまたは X の投稿説明に記載する。
