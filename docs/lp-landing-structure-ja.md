# 洋楽AIチャット ランディングページ構成案（ログイン前PR）

目的:
- ログイン前ユーザーに「ここは何の場か」を直感で伝える
- 人と人の交流が主役であることを明確にする
- AIは進行サポートと理解促進の伴走役だと伝える
- 日本における洋楽文化の活性化・再興という願いを打ち出す

画像生成の共通ルール（構図マンネリ防止）:
- 毎回、構図タイプを変える（同じレイアウトを繰り返さない）
- 同サイズの分割コマ（2x2、3x2、6タイル等）のコラージュ構図は禁止
- 画面内に説明UIを敷き詰めない。人物の表情・視線・仕草を主役にする
- 画面内デバイス表示は musicaichat のUIのみ（ビデオ会議グリッドは禁止）

7人ペルソナ（固定）:
- 20代男（オルタナティブ・ロック好き）
- 20代女（EDM好き）
- 30代男（Hip-hop好き）
- 40代女（R&B好き）
- 50代男（ハードロック好き）
- 60代男（British Invasion / classic pop-rock好き）
- 50代主婦（80s New Wave / POP好き）

7章立て（LPストーリー）:
1. Hero（世界観の入口）
2. ストーリー導入（ひとり聴き→つながり）
3. 魅力1（世代を超えた化学反応）
4. 魅力2（順番選曲と進行サポート）
5. 魅力3（AI解説と@質問）
6. 魅力4（80s New Wave / POPで広がる再発見）
7. クロージング（交流と再興への願い）

7カット主役ローテーション（ストーリー軸）:
- Hero: 30代男（Hip-hop好き）
- ストーリー導入: 20代男（オルタナティブ・ロック好き）

- 魅力3: 20代女（EDM好き）

### 【魅力 1】「音楽好き」が共鳴する、世代を超えた化学反応
※最重要ブロック（情報量を多めに配置）

#### キャッチ
「あなたの一曲が、誰かの新しい人生曲になる。」

#### 説明文（長め）
この場所では、20代から60代まで、音楽を愛する人が対等に語り合います。
同世代同士なら、当時の空気や思い出を共有できる。世代を超えれば、
自分が通ってこなかった時代やジャンルと出会える。

誰かの「推し曲」が、別の誰かにとっては“初めて心を奪われる一曲”になる。
それがこのコミュニティのいちばんの価値です。

リアルタイム同時視聴だからこそ、感情のタイミングまで重なる。
チャットの向こうにいる顔の見えない誰かと、同じ瞬間に震える体験は、
洋楽をただ消費するのではなく「共有する文化」へ変えてくれます。

そしてそれは、日本で洋楽をもう一度盛り上げる小さな火種になります。
