# スタート段階での運用費用

洋楽AIチャットを **スタート段階（MVP・個人・スモールスタート）** で運用する場合の費用を整理する。無料枠を最大限に使う前提。

---

## 1. 月額の目安（スタート時）

| 項目 | サービス | プラン | 月額（目安） | 備考 |
|------|----------|--------|--------------|------|
| ホスティング | Vercel | Hobby（無料） | **0円** | 月間10万PV程度まで無料。Next.js のデプロイ・本番URL。 |
| DB・認証 | Supabase | Free | **0円** | 500MB DB、50,000 レコード等。スタート時は十分。 |
| リアルタイム | Ably | Free | **0円** | 月間600万メッセージまで無料。100人同時接続程度なら収まる。 |
| AI | Google Gemini API | 従量課金（無料枠あり） | **0円〜500円** | 無料枠内で収まることも多い。利用量に応じて数百円程度になる可能性。 |
| 動画・曲メタ | YouTube IFrame API / oEmbed | 無料 | **0円** | API 利用料はかからない。 |
| **月額 合計** | | | **約 0円〜500円** | ほぼ無料枠で運用する想定。 |

---

## 2. 年額・初回のみ

| 項目 | 内容 | 費用（目安） |
|------|------|--------------|
| ドメイン | 候補 musicaichat.com 等を取得する場合 | **約 1,500円/年〜**（レジストラにより異なる） |
| 備考 | スタート時は Vercel の *.vercel.app で公開すればドメイン代は 0 円にできる。 | |

---

## 3. スタート段階で「ほぼ 0 円」にするための前提

- **Vercel**: Hobby のまま。Pro（有料）にはしない。
- **Supabase**: Free のまま。DB と認証のみ使用。
- **Ably**: Free のまま。チャット・同期再生のメッセージ数が月 600万以内。
- **Gemini API**: 無料枠を超えない範囲で利用。曲解説・相槌の呼び出しを「間引き」やキャッシュで抑えると、スタート時は 0 円で収まりやすい。
- **ドメイン**: 取得しない場合は 0 円。取得する場合は上記の年額のみ。

---

## 4. 費用が増えそうになるタイミング

| 要因 | 目安 | 対応例 |
|------|------|--------|
| アクセス増 | Vercel の帯域・ビルドが無料枠超過 | Pro（約3,000円/月）への移行を検討。 |
| DB 増大 | Supabase のストレージ・接続が Free の上限に | Pro（約3,700円/月）への移行を検討。 |
| 同時接続・メッセージ増 | Ably の無料枠超過 | 有料プラン（月 4,500円〜 等）を検討。 |
| AI 呼び出し増 | 金土夜の複数ルーム・多数参加で Gemini の利用量増 | 従量課金で月 500円〜1,500円程度まで増える可能性。キャッシュ・間引きで抑制。 |

---

## 5. まとめ（スタート段階）

| 項目 | 目安 |
|------|------|
| **月額運用費** | **約 0円〜500円**（無料枠中心。Gemini の利用量で数百円になる場合あり） |
| **年額** | ドメインを取得する場合 **約 1,500円/年〜**。取得しなければ 0 円。 |
| **方針** | 最初は無料枠で運用し、利用量・アクセスに応じて有料プランへ段階的に移行する。 |

※ 料金はサービス提供者の価格改定により変動するため、正式な金額は各サービスの公式ページで確認すること。
