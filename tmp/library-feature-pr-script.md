# ライブラリ機能 概要（PR用）

洋楽AIチャットのライブラリ機能は、  
**「聴きたい曲を、迷わず・探しやすく・このアプリ内で完結して選べる」**ための機能です。  

特に、  
- 選曲に迷う人  
- 好きなアーティストはいるが YouTube で見つけにくい人  
- スマホで YouTube とアプリを行き来せず、1画面で完結したい人  
に向いています。  

## 何が便利か（魅力）

1. **迷わず選べる**  
   アーティストや曲を一覧から選べるので、次に流す曲をすぐ決められます。  
   `[スクショ：ライブラリ一覧画面]`

2. **好きなアーティスト起点で探しやすい**  
   YouTube 検索に頼らず、アーティスト軸で曲へ辿れるため、見つからないストレスを減らせます。  
   `[スクショ：アーティスト→曲一覧画面]`

3. **アプリ内で選曲が完結**  
   洋楽AIチャット内で選曲まで完了でき、スマホでのアプリ切り替えが不要です。  
   `[スクショ：選曲完了〜再生導線]`

4. **洋楽ビギナーにやさしい**  
   「何を聴けばいいか分からない」状態でも、候補から直感的に選べるため、初めてでも使いやすい設計です。  
   `[スクショ：ビギナー向けにわかりやすい導線]`

## 一言まとめ（PR締め）

ライブラリ機能は、  
**「探す負担」と「選ぶ迷い」を減らし、洋楽AIチャット単体で快適に選曲できる、ビギナーにもやさしい体験**を提供します。
